Project Description
	- Project uses lwjgl library(It must have lwjgl2.0 library prior to run)
	- Randomly create Minecraft-like map that the user can use Cameracontroller class to move around and observe the map.
	- The terrain is randomized everytime the player execute the project.

	Control Key:

			W
		A 	S	D    , and  SHIFT(down), Space bar(up)



*Step by step for Setting up for lwjgl library(with Windows, using Eclipse) is provided.
