# VirtualMemorySimulator

	How to compile and run Assignment 1 from command line:
	
		1) Extract source code from compressed file.

		2) Make sure java jdk is installed correctly, "java -version" to check. (Version of java I am running: java version "1.8.0_171" java(TM) SE Runtime Environment (build 1.8.0_171-b11) Java HotSpot(TM) 64-Bit Server VM (build 25.171-b11, mixed mode))

		3) Change current directory to directory where the directory (outside of the source code) file is located in using "cd <path-to-directory>".

		4) To compile: "javac src/VMmain.java src/...", where ...= the rest of the java files.

		5) To run: "java VMmain test_1.txt".

		6) Check the created CSV file.
